# AWS EC2 + Nginx Mini Project

This mini project documents the steps and understanding required to deploy a basic Nginx web server on an AWS EC2 instance.

## Project Overview
- Launch an EC2 instance
- Configure security groups
- Install and run Nginx
- Verify application access

## Tools & Services
- AWS EC2
- Linux (Ubuntu)
- Nginx
- SSH

## High-Level Steps
1. Create and launch EC2 instance
2. Configure key pair and security group (port 22, 80)
3. Connect to EC2 using SSH
4. Install Nginx on the server
5. Start and enable Nginx service
6. Access Nginx via public IP

## Verification
- Check Nginx service status
- Open public IP in browser
- Confirm default Nginx page loads

## Learning Outcome
- Understanding cloud server setup
- Basics of Linux server management
- Importance of security groups and verification
